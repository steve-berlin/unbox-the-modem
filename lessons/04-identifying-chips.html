Part 4 — Identifying the Main Chips

Learn what each chip does — and how to look up your own.

TL;DR

A handful of chips do most of the work: a main processor (the brain), a memory chip, a permanent-storage chip, a networking chip, and an optical part for the fiber.

Each chip usually has tiny markings printed on top — a part number that names exactly what it is.

You read that number and search for it online to find the maker's datasheet, which describes the chip.

We teach the method, not this model's exact numbers — you will read those off your own board.

The main chips you'll meet

Open almost any small fiber modem and you'll find a few recurring roles. The exact parts vary by model, but the jobs are the same:

ChipWhat it does
Main processor (SoC)The brain — runs the operating system and everything else. "SoC" means system on a chip: one chip that combines the processor with several other functions.
Memory (RAM)Short-term working memory. Fast, but wiped when power is removed.
Permanent storageKeeps the operating system, settings and data even when unplugged.
Network / PHY chipHandles the Ethernet connection. "PHY" is the part that turns data into the electrical signals sent down the cable, and back again.
Optical partSends and receives the light in the fiber, converting between light and electrical signals.

On many devices the SoC does several of these jobs itself — a modern main processor often has the networking and even some storage built in. So don't be surprised if you find fewer separate chips than the list suggests.

How to read chip markings

Look at the top face of a chip. Printed on it — sometimes laser-etched and faint — is a set of letters and numbers. This usually includes the maker's logo, a part number that identifies the exact chip, and often a date and batch code you can ignore.

Tips for reading them:

  • Use good light held at an angle; faint etched text shows up when light rakes across it.
  • A magnifier or your phone camera's zoom helps a lot — take a sharp, close photo.
  • Copy the longest, most specific-looking string. That's usually the part number.
  • Note the maker's logo too; it narrows down what the number means.

Read markings only with the device unplugged, and mind static — hold the board by its edges, as in Part 3. You are looking, not touching the pins.

How to look a chip up

Once you have the part number, search for it online together with the word datasheet. A datasheet is the document the chip's maker publishes describing exactly what it is and how it works. From it you can usually learn the chip's role, how much memory or storage it holds, and how it connects to the rest of the board.

If the exact number gives nothing, search a shortened version — the first several characters often identify the family even when the full code is specific to one batch. Cross-check what you find against the chip's size and location on the board; a big central chip is almost certainly the main processor, not a tiny support part.

This course does not list the Nokia G-010G-P's exact chip numbers. Boards get revised and parts get swapped, so the only reliable source is the chip in front of you. Read the markings off your unit and look them up — that skill outlasts any list we could print.

Examples

Spotting the brain. One chip is clearly the largest, sits centrally, and has the most connections. You photograph its top, read the part number, search it with "datasheet," and confirm it's the main processor — with the networking built in.

The faint marking. A storage chip's text is almost invisible until you tilt a desk lamp across it. Suddenly the etched characters catch the light and you can read the number.

The shortened search. The full code returns nothing useful. You drop the last few characters, search again, and land on the chip family's page — enough to tell it's a memory chip and roughly how much it holds.

Try it yourself (safe)

  1. Find any old circuit board — an unused router, a broken gadget — or a clear close-up photo of one online.
  2. Pick the largest chip and read the text off its top face. Use raking light or a zoomed photo if it's faint.
  3. Search that part number online with the word "datasheet." See what role the chip turns out to have.

If you practise on the Nokia G-010G-P itself, keep it unplugged and handle it by the edges. This is a look-only exercise — you're reading markings, not connecting or removing anything.

Self-check: you read a chip's number but the search returns nothing. What next?

Search a shortened version — the first several characters usually name the chip's family even when the full code is batch-specific. Then cross-check against the chip's size and position on the board to confirm which role it plays.

Recap

  • A few chips do the main work: processor (SoC), memory, permanent storage, networking/PHY, and the optical part.
  • Modern processors often fold several of these jobs into one chip.
  • Chips carry printed part numbers you can read with good light and a close photo.
  • Search a part number with "datasheet" to learn what a chip is.
  • Read your own board's markings — don't rely on a printed list that may be out of date.
